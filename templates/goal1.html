Developing a Digital Tool

Motivation: During my studies I completed a coding course in which I created a digital maths learning tool for primary school children. This sparked my interest in how technology can create more personalised and engaging learning experiences. I find this topic challenging because integrating digital tools effectively requires both technical skill and a deep understanding of how children learn — especially in a diverse, international classroom where individual differences are significant. Positive feedback from my course tutor motivated me to explore this further in practice.

PDP Goal: I will develop and implement a digital maths learning tool that supports children in practising basic numeracy skills independently in order to create a learning environment that motivates individual learners and responds to individual differences.

Pico Project

After learning the basics of Python, I took on a hands-on project with the Raspberry Pi Pico. I programmed the integrated LED to create a functioning Morse code system, explored reading data from the onboard temperature sensor, and connected the Pico Pi to the internet to experiment with real-time data communication. This project challenged me to solve practical problems, think creatively, and combine software with hardware in innovative ways. Seeing my code come to life in a physical device was both rewarding and inspiring, and it strengthened my confidence in applying programming skills to tangible, real-world projects.
