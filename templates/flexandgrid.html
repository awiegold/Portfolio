Margins

Width

Height

Padding

Colour

Background Colour and Border

Background-color changes the color behind the content. Border creates a line around an element.

HTML Entities

& → This writes a normal & on the website. You must use & because & normally starts an HTML entity.

"" → This writes a double quotation mark " .

< → This writes the < symbol. In HTML, < normally starts a tag, so it must be written as <.

> → This writes the > symbol. It ends HTML tags, so you cannot type it directly inside text.

' '; → This creates a non-breaking space. The browser will not collapse or break this space. Example: Word   Word (three spaces).

'&euro'; → This writes the Euro sign € .

'& copy'; → This writes the copyright symbol © .

'&trade'; → This writes the trademark symbol ™ .

Grid Gallery

Static= The element stays in the normal document flow (the natural order on the page).
Relative= The element stays in the normal flow but you can shift it slightly using top, left, right, or bottom.
Absolute= The element is removed from the normal flow. It does not push other elements around. It positions itself relative to the closest parent that has position: relative (or absolute, fixed, sticky). If there is no such parent, it positions itself relative to the whole page.
Fixed= The element is positioned relative to the browser window. It stays in the exact same place even when the page is scrolled. Often used for sticky menus, chat bubbles, or floating buttons.
Sticky= This behaves like relative until you scroll. Then it becomes stuck to the top (or another chosen side).
Inherit= The element copies the position value of its parent. If the parent is relative, the child becomes relative. If the parent is static, the child becomes static.
